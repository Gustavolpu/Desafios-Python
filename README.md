# 🐍 Exercícios e Estudos de Python - Gustavo Guanabara

Este repositório foi criado para armazenar meus exercícios, desafios e anotações durante os estudos de Python através do curso do professor Gustavo Guanabara, do Curso em Vídeo.

## 📚 Sobre

Atualmente estou estudando os conceitos básicos da linguagem Python e praticando através dos exercícios propostos ao longo das aulas.

Meu objetivo é desenvolver uma base sólida em programação, lógica e boas práticas para evoluir para projetos mais avançados e automação.
